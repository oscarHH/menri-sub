Bienvenidos al repositorio menri-sub

Menri-sub herramienta para traducción de manga o cómics.

Este proyecto se crea por que aun no existe una herramienta que nos facilite la traducción de mangas o cómics. Siempre se tenia que hacer uso del editor de texto y un visor de imágenes para poder realizar la traducción, esto es algo molesto ya que son dos programas separados, por ejemplo al escribir no muestra un resaltado de sintaxis para distinguir las palabras, luego esta el visor de imágenes (no todos tienen soporte para el formato psd) al ser un programa aparte del editor de texto se tiene que pasar al visor de imágenes para poder cambiar de imagen y regresar al editor de texto.

Con todos estos problemas se creo la herramienta Menri-sub cuenta con las siguientes características

Características

  • Escrito en c++ con el framework QT 5
  • Resaltado de sintaxis
  • Soporta varios formatos de imagen(jpg,png,bmp,psd)
  • Manejo de archivos comprimidos
  • Zoom
  • Rota imágenes
  • vista en miniatura de imágenes
  • Multiplataforma y algunas funciones que se estarán agregando
